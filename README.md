# LSD - Little Sticky Destroyer

![LSD - Screenshot](https://raw.github.com/paranoiax/LSD/master/screenshot.jpg)

## What is LSD?

LSD is a physics based volleyballgame written in lua using the löve game engine.
It is currently under development.

## How to play?

Shoot the ball onto the grey platforms.
As you leave a platform, it get's destroyed - Goal is to destroy all of them.
Avoid the red walls.
Beware of the "Timer of Death©®™" on the bottom

## Controls

* Shoot:   "LEFT MOUSE BUTTON"
* Retry:   "R" , "ENTER" or "SPACE"
* Exit / Menu:   "ESCAPE"
