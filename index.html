LSD - Little Sticky Destroyer

What is LSD?

LSD is a physics based arcade / action / platform game written in lua using the löve game engine. It is currently under development.

How to play?

Shoot the ball onto the grey platforms. You will always stick to them. As you leave a platform, it get's destroyed - Goal is to destroy all of them. Avoid the red walls. Beware of the timer on the bottom. Everytime you stick to a platform, the timer starts to count down. If it reaches zero, you will die. The timer resets, as soon as you leave a platform.

Controls

  • Shoot: "LEFT MOUSE BUTTON"
  • Retry: "R" , "ENTER" or "SPACE"
  • Exit / Menu: "ESCAPE"
